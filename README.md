# 📖 Project Overview

The **Bank Loan Analysis Dashboard** is an interactive business intelligence solution developed using **Microsoft Power BI**. The dashboard provides a comprehensive analysis of loan applications, customer demographics, financial performance, loan distribution, and credit risk.

The objective of this project is to transform raw loan data into meaningful business insights that help banks and financial institutions monitor lending performance, understand customer behavior, identify potential risks, and support data-driven decision-making.

Through interactive visualizations, dynamic filters, and key performance indicators (KPIs), users can easily explore lending trends, customer characteristics, loan purposes, credit scores, and repayment-related insights.

---

# 🎯 Project Objectives

- Analyze overall loan application performance.
- Monitor total loan amount and average loan amount.
- Evaluate customer income and demographic information.
- Analyze loan purposes and home ownership.
- Identify customer borrowing patterns.
- Evaluate credit score distribution and lending risk.
- Generate actionable business insights for financial decision-making.

# 📈 Key Performance Indicators (KPIs)

| KPI | Description |
|------|-------------|
| Total Loan Applications | Total number of loan requests received |
| Total Loan Amount | Overall value of approved/requested loans |
| Average Loan Amount | Average loan amount per customer |
| Average Interest Rate | Average interest charged on loans |
| Average Credit Score | Average borrower credit score |
| Average Customer Income | Average annual income of applicants |

# 📌 Business Insights

- Customers with higher credit scores generally qualify for larger loan amounts.
- Loan purpose significantly influences borrowing trends.
- Employment experience has a positive impact on average customer income.
- Home ownership categories show different borrowing behaviors.
- Interest rate ranges affect overall loan distribution.
- Customer demographics provide valuable insights into lending patterns.
- Credit history and previous defaults help identify potential lending risks.
