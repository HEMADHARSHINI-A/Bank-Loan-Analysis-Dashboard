# 📖 Project Overview

The **Bank Loan Analysis Dashboard** is an interactive Power BI project that provides valuable insights into loan applications, borrower demographics, financial performance, and lending risks. This dashboard helps banks and financial institutions monitor loan trends, evaluate customer profiles, and support data-driven decision-making through dynamic visualizations.

The dashboard enables users to analyze loan amounts, interest rates, customer income, credit scores, loan purposes, and approval trends using interactive charts, KPI cards, and slicers.

---

# 🎯 Objectives

- Analyze overall loan applications and loan amounts.
- Monitor customer income and credit scores.
- Understand borrower demographics.
- Evaluate loan purposes and home ownership.
- Analyze loan approval and risk patterns.
- Generate meaningful business insights for better financial decisions.

# 💡 Business Insights

- Loan applications are distributed across different loan purposes, helping identify customer borrowing trends.
- Customers with higher credit scores generally qualify for larger loan amounts.
- Employment experience and annual income influence loan eligibility and borrowing capacity.
- Home ownership categories provide insights into customer financial stability.
- Credit history and previous loan defaults help identify high-risk applicants.

# 📌 Dashboard Benefits

- Provides a comprehensive overview of loan performance.
- Helps identify customer borrowing patterns.
- Supports credit risk analysis.
- Enables interactive filtering using slicers.
- Assists financial institutions in making data-driven lending decisions.
